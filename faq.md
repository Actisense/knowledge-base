# Frequently Asked Questions

Answers to common questions about Actisense products, the SDK, and integrating Actisense devices into your projects.

## General

**Q: Where can I find official documentation for Actisense products?**

A: Full product manuals, technical guides, and release notes are available at [actisense.com/support](https://www.actisense.com/support/).

---

**Q: How do I get started with integrating an Actisense device into my project?**

A: The best starting point is the [SDK Repository](https://github.com/Actisense/SDK), which contains libraries and examples for connecting to Actisense devices across multiple platforms and languages.

---

**Q: What platforms and languages does the Actisense SDK support?**

A: The SDK supports C, C++, and Python on Windows, Linux, and macOS. See the [SDK Repository](https://github.com/Actisense/SDK) for the full list and platform-specific instructions.

---

**Q: How do I report a bug or request a new feature?**

A: Please open an issue in the relevant GitHub repository. For SDK issues, use the [SDK issue tracker](https://github.com/Actisense/SDK/issues). For knowledge base content, use the issue tracker here.

---

**Q: How do I contact Actisense engineering support?**

A: Reach out via [actisense.com/contact](https://www.actisense.com/contact/).

---
