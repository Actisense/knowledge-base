## General Tips

- Always ensure you are running the latest firmware for your Actisense device. Check [actisense.com/support](https://www.actisense.com/support/) for the latest releases.
- Check the [SDK Repository](https://github.com/Actisense/SDK) for known issues and updates.
- If you have found a fix for an issue not listed here, please consider [contributing](../CONTRIBUTING.md)!
